# java-library-system
A simple Java OOP project that demonstrates inheritance, polymorphism, and method overriding using a library book system.

##  Features

* Create different types of books
* Display book details
* Show availability status
* Demonstrates runtime polymorphism



##  Concepts Used

* Classes and Objects
* Inheritance
* Method Overriding
* Constructors
* Polymorphism
* super keyword

 

##  Project Structure

* Book (Parent Class)
* Novel (Child Class)
* TextBook (Child Class)
* Main (Runs the program)

 


##  Example Output

```

ID: 101

```

##  Learning Outcome

This project helps understand how OOP concepts are used to model real-world systems like a library.
